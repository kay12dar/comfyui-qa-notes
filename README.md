# ComfyUI QA Notes

Research into how generation parameters affect image quality in ComfyUI.
Documents parameter testing methodology, findings, and defect patterns.

## Goal
Understand how to evaluate and tune generative image output — useful for
QA of AI products that use image/video generation pipelines.

## Files
- `parameter_research.md` — how each parameter affects output quality
- `defect_catalog.md` — common generation defects with descriptions
- `test_log.md` — sample test runs with settings and observations
